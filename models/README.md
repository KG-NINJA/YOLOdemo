# Models

このディレクトリに YOLOv8 の ONNX モデルを配置してください。推奨ファイル名:

- `yolov8n-quantized.onnx` (CPU/WASM 推奨・デフォルトで使用)
- `yolov8n.onnx` (量子化版が入手できない場合のフォールバック)
- `yolov8m.onnx`
- `yolov8l.onnx`

ファイルサイズが大きいためリポジトリには含めていません。Ultralytics 公式のエクスポート機能や事前量子化済みモデルをダウンロードして配置してください。`npm run fetch:yolo` を実行すると量子化版の取得を試み、失敗した場合は通常版を自動取得します（ネット接続が必要）。

※ 量子化モデルが見つからない場合はローカルの `yolov8n.onnx` に自動で切り替え、それも無い場合のみブラウザ実行時に Hugging Face (`onnx-community` コレクション) から同等モデルをダウンロードするフォールバックを備えています。ローカル読み込みでは `models/...` に加えて `./models/...` や `/models/...` も順に試し、ホスティングパスの違いによる 404 を避けます。
